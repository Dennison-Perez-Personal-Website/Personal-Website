My CCS learning path

I have learned and continue learning more about Cascading Style Sheets ! There are three ways to insert CSS! The first way is with inline styling. You style the element you are using to make it "pretty" using CSS. For example -div style="font: bold;"- will make all font bold in the -div- or whichever else container it is used in. The second way to style would be with an internal style setup. Above the body section, in the head, of your HTML setup page you add your styling with a -style- element section. For example if I were to say all - h1 - I want to be bold I can accomplish this by -style- then choose an attribute and close the tag.
